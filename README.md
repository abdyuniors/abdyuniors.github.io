## 👨‍💻 Tentang Repositori Ini

Selamat datang di *source code* portofolio pribadi saya! Website ini dirancang untuk memamerkan proyek-proyek unggulan, perjalanan karir, dan keahlian teknis saya dalam menjembatani ekosistem **Mobile Native** dan **Web Development**.

Portofolio ini mengedepankan performa tinggi dan desain responsif, dibangun dengan teknologi *frontend* modern:

*   ⚡ **Framework:** [Astro](https://astro.build/) - Untuk kecepatan maksimal dan *Static Site Generation* (SSG).
*   🎨 **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) - Menggunakan integrasi Vite terbaru untuk desain utilitas yang ringan dan dinamis.
*   ⚙️ **Deployment:** GitHub Actions & GitHub Pages.
